Danmaku Demo

A browser mini-stage inspired by the classic Touhou shooters — focus shot, graze, bombs, and a Tenshi-flavoured spellcard finale. Original code; not a port.

Controls

  • ←↑↓→ / WASD Move
  • Shift Focus (slow + hitbox)
  • Z / Space Shoot · start
  • X Bomb
  • P / Esc Pause
  • Touch: drag to move, auto-focus fire

Survive fairy waves, then face a three-phase celestial boss pattern. Graze bullets for points. This is a fan-made homage — mechanics only.
